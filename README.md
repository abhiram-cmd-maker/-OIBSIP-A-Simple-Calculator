# Basic Calculator using HTML, CSS, and JavaScript

## Overview

This project is a simple **web-based calculator** built using **HTML, CSS, and JavaScript**. It performs basic mathematical operations such as:

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)

The calculator provides an interactive user interface where users can click buttons to enter numbers and operators, view input on a display screen, and calculate results instantly.

---

## Features

* Interactive calculator interface
* Display screen for showing input and output
* Basic arithmetic operations
* Responsive button layout using CSS Grid
* Clear/Delete functionality
* Real-time user interaction with JavaScript event handling

---

## Technologies Used

* **HTML** – Structure of the calculator
* **CSS** – Styling and grid layout for buttons
* **JavaScript** – Logic and functionality

---


---

## How It Works

### HTML

HTML creates the structure of the calculator:

* Display screen
* Number buttons
* Operator buttons
* Equal button
* Clear button

### CSS

CSS is used for:

* Styling the calculator
* Aligning buttons using **CSS Grid**
* Creating responsive layout

### JavaScript

JavaScript adds calculator functionality using:

* **Event Listeners** to detect button clicks
* **Operators** for mathematical calculations
* **If-else statements** for logic control
* **Loops** where necessary for button handling

---

## Functional Logic

1. User clicks a number or operator button
2. Input appears on the display screen
3. JavaScript stores the expression
4. Pressing `=` evaluates the result
5. Result is shown on the display
